# 🛒 E‑Commerce Website (MERN Stack)

A modern, responsive **e‑commerce platform** built using the MERN stack (MongoDB, Express.js, React, Node.js).  
This project demonstrates full‑stack development skills with secure authentication, product management, and payment workflows.

---

## 🚀 Features
- **Product Catalog** – Browse items with categories, filters, and search functionality.
- **Shopping Cart & Checkout** – Add/remove products, view totals, and complete orders.
- **User Authentication** – Secure sign‑up/login with JWT and profile management.
- **Payment Gateway Integration** – Razorpay sandbox (or other gateways) for safe transactions.
- **Order Management** – Track and manage customer purchases.
- **Responsive Design** – Optimized for desktop, tablet, and mobile devices.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, HTML5, CSS3, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ODM)  
- **Authentication:** JWT, bcrypt  
- **Payment Integration:** Razorpay sandbox  
- **Tools & Workflow:** VS Code, GitHub, npm scripts, environment configuration
