🛒 User Guide – Website

1️⃣ Home Page

Displays featured products, categories, search bar and navigation menu.

2️⃣ User Registration

Click Sign Up → Enter details → Account created.

3️⃣ User Login

Login using registered email and password.

4️⃣ Browse Products

Select category → Apply filters → View product details.

5️⃣ Product Details

Check description, price, reviews → Add to cart or Buy now.

6️⃣ Add to Cart

Update quantity or remove items → Proceed to checkout.

7️⃣ Checkout & Payment

Enter shipping → Choose payment → Confirm order.

8️⃣ Order Tracking

Go to Track Order → View delivery status.

9️⃣ Logout

Click profile → Logout securely.
